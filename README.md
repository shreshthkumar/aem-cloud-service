# Adobe AEM as a Cloud Service Demo Site

This project demonstrates the use of Adobe Experience Manager (AEM) as a Cloud Service with a sample site. The archetype provides a quick start for developers to build applications and custom components leveraging AEM Cloud capabilities.

## Getting Started

To get started with the demo site:

1. **Clone the repository**
   ```bash
   git clone https://github.com/shreshthkumar/aem-cloud-service.git
   cd aem-cloud-service
   ```

2. **Install Dependencies**
   - Ensure you have the necessary tools and dependencies installed. Refer to the official AEM documentation for a complete list.

3. **Run the Demo Site**
   - Follow the steps provided in the documentation to run the demo site locally or deploy it to AEM as a Cloud Service.

## Project Structure

- **/content** - Contains all site-related content.
- **/ui.apps** - Contains the client libraries, components, and templates used in the project.
- **/ui.content** - Contains site content and structure.
